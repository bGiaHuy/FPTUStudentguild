# Map Sources

This folder is intended for raw mapping source files.
Specifically, it will contain the manually traced SVG files representing accurate room boundaries based on official floor plan PNGs.

## Expected Files
- `delta_floor1_traced.svg`: The manually traced SVG for Floor 1 using `floor1.png` as a background.
- `delta_floor3_traced.svg`: The manually traced SVG for Floor 3 using `floor3.png` as a background.

*Note: Floor 2 and Floor 4 remain without geometry until official images are provided.*

## Tracing Workflow
Please refer to `../map_drafts/tracing_workflow.md` for exact instructions on how to trace the rooms.
